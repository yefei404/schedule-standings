# 赛程 · 积分 · 淘汰赛

多小组赛事管理工具，支持赛程录入、积分榜统计、世界杯淘汰赛对阵、数据导入导出与打印，适用于 iPad 现场使用。

## 在线体验

GitHub Pages: `https://yefei404.github.io/schedule-standings/`

## 功能特性

- **三模式切换**：赛程表 / 积分榜 / 淘汰赛赛程
- **赛程管理**：添加比赛、录入比分、上传队徽、删除/清空、加载示例
- **多小组积分榜**：支持多个小组，每组独立队名、队徽、胜平负、进失球、积分
- **淘汰赛对阵**：32强 → 16强 → 8强 → 半决赛 → 三四名 → 决赛，可编辑队名/比分/队徽
- **自动保存**：localStorage 持久化（`tournament_data_v2`），修改后 300ms 防抖保存，刷新不丢失
- **导入导出**：JSON 格式备份与恢复（含赛程、积分榜、淘汰赛）
- **打印支持**：一键打印当前视图
- **快捷键**：赛程模式下 Ctrl/Cmd + Enter 快速添加比赛
- **iPad 桌面图标**：保存到主屏幕时显示蓝色「8」图标

## 设计说明

### 界面结构

```
┌──────────────────────────────────────────────┐
│  ⚽ [赛事名称]              [添加][示例][清空]  │
├──────────────────────────────────────────────┤
│  📋 赛程表  │  📊 积分榜  │  🏆 淘汰赛赛程     │
├──────────────────────────────────────────────┤
│  （当前面板内容）                              │
├──────────────────────────────────────────────┤
│  统计信息          [导出 JSON][导入 JSON][打印] │
└──────────────────────────────────────────────┘
```

### 淘汰赛默认结构

| 轮次 | 场次 |
|------|------|
| 32强 | 16 |
| 16强 | 8 |
| 8强 | 4 |
| 半决赛 | 2 |
| 三四名 | 1 |
| 决赛 | 1 |

| 项目 | 方案 |

| 导入 | 读取 JSON 文件覆盖 |
