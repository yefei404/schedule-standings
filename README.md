# 赛程表 & 积分榜

多小组赛事管理工具，支持赛程录入、积分榜统计、数据导入导出与打印，适用于 iPad 现场使用。

## 在线体验

GitHub Pages: `https://yefei404.github.io/schedule-standings/`

## 功能特性

- **双模式切换**：赛程表 / 积分榜 Tab 切换
- **赛程管理**：添加比赛、录入比分、上传队徽、删除/清空、加载示例
- **多小组积分榜**：支持多个小组，每组独立队名、队徽、胜平负、进失球、积分
- **自动排序**：积分榜按积分 → 净胜球 → 进球数排序
- **数据持久化**：localStorage 自动保存，刷新不丢失
- **导入导出**：JSON 格式备份与恢复
- **打印支持**：一键打印当前视图
- **快捷键**：赛程模式下 Ctrl/Cmd + Enter 快速添加比赛
- **iPad 桌面图标**：保存到主屏幕时显示蓝色「8」图标

## 设计说明

### 界面结构

```
┌─────────────────────────────────────┐
│  🏆 [赛事名称]          [导入][导出]  │
├─────────────────────────────────────┤
│  📋 赛程表  │  📊 积分榜              │
├─────────────────────────────────────┤
│  （当前面板内容）                      │
└─────────────────────────────────────┘
```

### 赛程表

- 每场比赛：左队 / 比分 / 右队，中间分隔线
- 队徽 3:2 比例，点击上传图片
- 比分可编辑，支持添加、删除、清空

### 积分榜

- 多小组结构，每组有独立名称
- 每队字段：场次、胜、平、负、进球、失球、净胜球、积分
- 支持添加/删除小组、添加/删除球队
- 修改数据后自动重算排序

### 数据存储

| 项目 | 方案 |
|------|------|
| 存储 | localStorage |
| 导出 | JSON 文件下载 |
| 导入 | 读取 JSON 文件覆盖 |

### 技术栈

| 层面 | 方案 |
|------|------|
| 结构 | 原生 HTML5 |
| 样式 | 原生 CSS |
| 逻辑 | 原生 JavaScript（IIFE 单文件） |
| 部署 | GitHub Pages |
